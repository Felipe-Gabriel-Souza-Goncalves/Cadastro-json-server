## ✏️ Descrição:
Simulação de cadastro com json-server e localStorage, feito para estudo das tecnologias citadas

## ⚙️ Funcionalidades:
- Criar usuário
- Logar com o usuário
- Deletar o usuário
- Entrar como anônimo
- Acessar links externos para outros projetos
